Detailed Description

Parameters for active stress models.

This class stores all the parameters related to active stress, including e.g. the name of the specific selected model. The parameters specific to an individual model are managed by the class ActiveStressModelParameters, of which this class owns an instance for every registered model.

In the XML file, this class corresponds to the <Active_stress> element.
