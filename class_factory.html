Detailed Description

template<class BaseType>
class Factory< BaseType >

Abstract self-registering factory.

This class gives a way to create a dynamic register of classes derived from a certain base class, and then instantiate them by name. To be compatible with this factory, the derived classes must be default constructibleß.

It combines the factory and singleton patterns. There should always exist only one instance of this class, which cannot be accessed directly but only manipulated through the static methods of this class.

To register a class into the factory, you can call the register_child static method, passing a class derived from BaseType as template argument and a label for it as argument. A shortcut for this is to use the macro REGISTER_IN_FACTORY.

Iterate through the registered classes.

For every registered class derived from BaseType, creates a dummy instance of it, and then calls the provided function on that instance. All the dummy instances are destroyed after the function call.

Registered derived classes.
