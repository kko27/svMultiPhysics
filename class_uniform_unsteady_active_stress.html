Detailed Description

Uniform and time dependent active stress model.

Defines an active tension that is constant in space but time dependent, i.e.

\[ \Tact(t, \calcium, \fiberstretch, \fiberstretchrate, \astressstate) = g(t)\;, \]

where \(g(t)\) is a user-defined function of time.
